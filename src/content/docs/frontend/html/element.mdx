## 功能和用途分类

### 结构性元素（Structural Elements）

这些元素定义了页面的结构和布局，帮助浏览器和开发者理解网页的组织方式

- `<html>` - 整个 HTML 文档的根元素
- `<head>` - 包含页面的元数据，如标题、链接到 CSS 文件、脚本等
- `<body>` - 页面的主体部分，包含所有可视化的内容
- `<header>` - 定义页面或部分内容的头部，通常包括导航、标题等
- `<footer>` - 定义页面或部分内容的底部，通常包含版权信息、联系方式等
- `<section>` - 定义页面的一个区域或章节，通常包含一个主题内容
- `<article>` - 定义独立的内容单元，如博客文章、新闻等
- `<nav>` - 定义页面中的导航区域，包含链接
- `<aside>` - 定义与页面内容关联但相对独立的部分，如侧边栏或广告
- `<main>` - 定义页面的主要内容，通常包含页面的核心信息

### 文本内容元素（Text Content Elements）

这些元素用于格式化文本内容，包括标题、段落、列表等

`<h1>`, `<h2>`, `<h3>`, ..., `<h6>`：表示标题，`<h1>` 是最重要的标题，`<h6>` 是最不重要的

- `<p>` - 定义段落
- `<blockquote>` - 表示长引用
- `<ul>` `<ol>` `<li>` - 无序列表（`<ul>`）、有序列表（`<ol>`）和列表项（`<li>`）
- `<a>` - 定义超链接
- `<strong>` - 强调文本（通常显示为加粗）
- `<em>` - 强调文本（通常显示为斜体）
- `<b>` - 加粗文本，但没有语义上的强调，建议使用 `<strong>` 代替
- `<i>` - 斜体文本，但没有语义上的强调，建议使用 `<em>` 代替
- `<br>` - 换行符
- `<span>` - 行内元素，通常用于样式或标记一小段文本

### 媒体元素（Media Elements）

这些元素用于在页面上嵌入和显示图像、视频、音频等媒体内容

- `<img>` - 用于插入图像
- `<audio>` - 用于嵌入音频内容
- `<video>` - 用于嵌入视频内容
- `<source>` - 用于指定 `<audio>` 或 `<video>` 的资源文件
- `<track>` - 用于提供视频或音频的字幕、描述或章节

### 表单元素（Form Elements）

这些元素用于创建表单，允许用户输入数据

- `<form>` - 定义表单
- `<input>` - 定义用户输入的控件，可以是文本框、按钮、复选框等
- `<textarea>` - 定义多行文本输入框
- `<select>` - 定义下拉菜单
- `<option>` - 定义下拉菜单中的选项
- `<button>` - 定义按钮
- `<label>` - 定义表单控件的标签
- `<fieldset>` - 将表单控件分组，通常与 `<legend>` 一起使用
- `<legend>` - 为 `<fieldset>` 提供一个标题

### 表格元素（Table Elements）

这些元素用于创建表格，组织结构化数据

- `<table>` - 定义表格
- `<tr>` - 定义表格行
- `<td>` - 定义表格单元格
- `<th>` - 定义表格的表头单元格
- `<thead>` - 定义表格头部部分
- `<tbody>` - 定义表格的主体部分
- `<tfoot>` - 定义表格的脚部部分

### 链接和引用元素（Link and Reference Elements）

这些元素主要用于创建链接和引用外部资源

- `<a>` - 定义超链接
- `<link>` - 用于在 `<head>` 中引入外部资源（如 CSS 文件）
- `<meta>` - 提供页面的元数据，如字符集、作者信息等

### 脚本元素（Script Elements）

这些元素用于定义脚本内容或引入外部脚本文件

- `<script>` - 定义 JavaScript 脚本
- `<noscript>` - 用于提供在浏览器不支持脚本时显示的内容

## 根据元素的显示行为分类

HTML 元素还可以根据其显示行为分为两大类：块级元素和行内元素

### 块级元素（Block-level elements）

块级元素通常占据一整行，并且可以容纳其他块级元素或行内元素。它们从新的一行开始显示，宽度通常会自动填满容器的宽度

常见的块级元素包括：

`<div>`

`<h1>` 到 `<h6>`

`<p>`

`<ul>`, `<ol>`, `<li>`

`<section>`

`<article>`

`<footer>`

`<header>`

### 行内元素（Inline elements）

行内元素不会占据整行，它们只占据自身内容的宽度，并且可以在一行内与其他行内元素一起显示

常见的行内元素包括：

`<span>`

`<a>`

`<strong>`

`<em>`

`<img>`

`<b>`

`<i>`

`<code>`

### 自闭合元素（Self-closing elements）

自闭合元素是指那些不需要结束标签的元素。它们通常以 `/>` 结尾

常见的自闭合元素包括：

- `<img>` - 插入图像
- `<br>` - 换行
- `<hr>` - 水平分隔线
- `<input>` - 输入控件
- `<meta>` - 定义元数据
- `<link>` - 用于链接外部资源

### 根据内容类型分类

有些元素根据其内容类型进行分类

常见的内容类型包括：

结构化内容元素：如 `<header>`, `<footer>`, `<section>`, `<article>` 等，这些元素用于构建文档的结构

文本内容元素：如 `<h1>` 到 `<h6>`, `<p>`, `<strong>`, `<em>`, `<span>`, `<blockquote>` 等，用于格式化和显示文本

嵌套内容元素：如 `<a>`, `<img>`, `<audio>`, `<video>`, `<iframe>` 等，用于嵌入其他媒体或资源

表单元素：如 `<form>`, `<input>`, `<select>`, `<button>` 等，用于创建交互式表单
