## 压缩包安装

> [下载地址](https://nginx.org/en/download.html)

<Steps>

1.  解压下载后的压缩包
2.  安装依赖

    ```bash
    apt install -y libpcre3 libpcre3-dev zlib1g-dev openssl libssl-dev
    ```

3.  编译

    ```bash
    ./configure --prefix=<nginx编译后文件存放的位置>
    make && make install
    ```

4.  运行

    ```bash
    nginx -c <nginx配置文件>
    ```

</Steps>
