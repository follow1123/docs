## 安装

默认已经安装，如果没有则参考[旧版 WSL 的手动安装步骤](https://learn.microsoft.com/zh-cn/windows/wsl/install-manual)

### 安装Linux发行版

```batch
REM 查询可用的发行版
wsl -l -o

REM 安装Debian
wsl --install -d Debian

REM 查看已安装的发行版信息
wsl -l -v

REM 关闭wsl
wsl --shutdown
```

## 配置

### `wsl.conf` 文件

> [官方文档](https://learn.microsoft.com/zh-cn/windows/wsl/wsl-config#wslconf)

wsl 系统内部配置，位于虚拟机系统的 `/etc/wsl.conf`

```ini
[boot]
systemd=true # 启用 systemd

[interop]
enabled = true # 允许调用 windows 的程序
appendWindowsPath = false # 禁止加载 Windows 的 PATH 环境变量
```

### `.wslconfig` 文件

> [官方文档](https://learn.microsoft.com/zh-cn/windows/wsl/wsl-config#wslconfig)

wsl 相关配置，位于宿主机的 `$ENV:USERPROFILE\.wslconfig`

用于配置虚拟机内核个数，内存大小等，一般使用 `WSL Settings` 程序修改

win + q 搜索 `WSL Settings` 打开
