> 详细参考 `:h jumplist`

| 模式 | 按键                   | 描述             |
| ---- | ---------------------- | ---------------- |
| `C`  | `:jumps`               | 打开跳转列表     |
| `C`  | `:clearjumps`          | 清空跳转列表     |
| `N`  | <Kbd linux="ctrl+i" /> | 跳转到较旧的位置 |
| `N`  | <Kbd linux="ctrl+o" /> | 跳转到较新的位置 |
