> 详细参考 `:h changelist`

| 模式 | 按键                | 描述                 |
| ---- | ------------------- | -------------------- |
| `C`  | `:changes`          | 打开变更列表         |
| `N`  | <Kbd linux="g+;" /> | 跳转到较旧修改的位置 |
| `N`  | <Kbd linux="g+," /> | 跳转到较新修改的位置 |
