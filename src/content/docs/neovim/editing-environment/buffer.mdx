| 模式      | 按键                  | 描述                          |
| --------- | --------------------- | ----------------------------- |
| `Normal`  | <Kbd linux="ctrl+6"/> | 切换到上次打开的 buffer       |
| `Command` | `:ls`                 | 显示当前所有打开的 buffer     |
| `Command` | `:b#`                 | 和 <Kbd linux="ctrl+6"/> 相同 |
