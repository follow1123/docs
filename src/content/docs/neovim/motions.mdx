## 水平移动

| 模式 | 按键                                           | 描述                                                                                          |
| ---- | ---------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `N`  | <Kbd linux="w" />/<Kbd linux="W" />            | 光标右移动一个词/包含词周围的符号                                                             |
| `N`  | <Kbd linux="b" />/<Kbd linux="B" />            | 光标左移动一个词/包含词周围的符号                                                             |
| `N`  | <Kbd linux="0" />                              | 光标移动到行首，包含空白字符                                                                  |
| `N`  | <Kbd linux="^" />                              | 光标移动到行首, 不包含空白字符                                                                |
| `N`  | <Kbd linux="$" />                              | 光标移动到行尾                                                                                |
| `N`  | <Kbd linux="f" />/<Kbd linux="F" /> + `{char}` | 向左或向右搜索字符，光标移动该字符上                                                          |
| `N`  | <Kbd linux="t" />/<Kbd linux="T" /> + `{char}` | 向左或向右搜索字符，光标移动该字符周围                                                        |
| `N`  | <Kbd linux=";" />                              | 重复<Kbd linux="f" />/<Kbd linux="F" />/<Kbd linux="t" />/<Kbd linux="T" />按键搜索字符       |
| `N`  | <Kbd linux="," />                              | 反方向重复<Kbd linux="f" />/<Kbd linux="F" />/<Kbd linux="t" />/<Kbd linux="T" />按键搜索字符 |

## 垂直移动

| 模式 | 按键                   | 描述                     |
| ---- | ---------------------- | ------------------------ |
| `N`  | <Kbd linux="ctrl+u" /> | 向上移动半页             |
| `N`  | <Kbd linux="ctrl+d" /> | 向下移动半页             |
| `N`  | <Kbd linux="ctrl+f" /> | 向上移动一页             |
| `N`  | <Kbd linux="ctrl+b" /> | 向下移动一页             |
| `N`  | <Kbd linux="H" />      | 光标移动到当页的头部     |
| `N`  | <Kbd linux="M" />      | 光标移动到当页的中间     |
| `N`  | <Kbd linux="L" />      | 光标移动到当页的尾部     |
| `N`  | <Kbd linux="{" />      | 向上移动一段             |
| `N`  | <Kbd linux="}" />      | 向下移动一段             |
| `N`  | <Kbd linux="g+g" />    | 光标移动到文件的第一行   |
| `N`  | <Kbd linux="G" />      | 光标移动到文件的最后一行 |

## 其他

| 模式 | 按键                | 描述                                   |
| ---- | ------------------- | -------------------------------------- |
| `N`  | <Kbd linux="z+z" /> | 光标不动，光标下的内容移动到窗口的中间 |
| `N`  | <Kbd linux="%" />   | 光标移动到另一半大、中、小括号上       |
