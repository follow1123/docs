内置文件管理插件

> 详细参考 `:h netrw`

| 模式 | 按键        | 描述                       |
| ---- | ----------- | -------------------------- |
| `C`  | `:E`        | 打开文件管理器 buffer      |
| `C`  | `:Rexplore` | 返回上次打开的文件管理器   |
| `C`  | `:Lexplore` | 开启或关闭左测的文件管理器 |

进入 `netrw` 窗口内相关快捷键

| 按键                | 描述                    |
| ------------------- | ----------------------- |
| <Kbd linux="%" />   | 新建文件                |
| <Kbd linux="d" />   | 新建目录                |
| <Kbd linux="D" />   | 删除选中的文件/目录     |
| <Kbd linux="R" />   | 重命名（移动）文件/目录 |
| <Kbd linux="-" />   | 返回上一级目录          |
| <Kbd linux="i" />   | 切换不同的显示风格      |
| <Kbd linux="I" />   | 切换是否显示横幅        |
| <Kbd linux="r" />   | 反转排序顺序            |
| <Kbd linux="g+h" /> | 快速隐藏/显示点文件     |
