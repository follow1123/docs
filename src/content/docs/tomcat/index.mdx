## 安装

Windows 下使用压缩包安装

<Steps>

1. 在官网下载指定压缩包，官网左侧 `Download` 处选择版本

2. 解压到指定目录

3. 配置环境变量 `CATALINA_HOME`

4. 将 `%CATALINA_HOME%\bin` 添加到 `PATH` 环境变量

</Steps>

## 目录

安装Tomcat后的目录格式

- `bin/` - 存放启动和停止脚本
- `conf/` - 存放配置文件，控制 Tomcat 的行为
- `lib/` - 存放 Tomcat 运行所需的 Java 类库
- `logs/` - 存放运行时生成的日志文件
- `webapps/` - 存放部署的 Web 应用
- `work/` - 存放 Tomcat 的临时工作文件
- `temp/` - 存放临时文件
