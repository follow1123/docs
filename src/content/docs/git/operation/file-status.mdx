Git 仓库内可以分为：**工作目录** **暂存区** **本地仓库** **远程仓库**这几个区域，其中的文件可以分为：**未跟踪** **已暂存** **未修改** **已修改**这几个状态

![文件的状态](@assets/git/git-file-status.drawio.png)

- **未跟踪** - 仓库内新建的文件
- **已暂存** - 使用 `git add` 命令添加的文件
- **未修改** - 提交到**本地仓库**的文件
- **已修改** - 修改**本地仓库**内文件后的状态
