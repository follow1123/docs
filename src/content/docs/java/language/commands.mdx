JDK 自带工具

| 命令           | 说明                             |
| -------------- | -------------------------------- |
| `jps`          | 查看所有 java 进程信息           |
| `jps -l`       | 显示 main 方法                   |
| `jps -v`       | 显示详细信息                     |
| `jstack <pid>` | 查看指定 java 进程的所有线程信息 |
| `jconsole`     | 图形化界面，查看 jvm 信息        |
