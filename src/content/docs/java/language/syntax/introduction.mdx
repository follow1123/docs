## Hello World

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```

```bash
# 编译
javac HelloWorld.java

# 运行
java HelloWorld
```

## 注释

```java
/**
 * 文档注释
 * @author a
 * @version 1.0
 */
public class Comments {

    /*
    多行注释
    多行注释
    多行注释
    */
    public static void main(String[] args) {
        // 单行注释
        System.out.println("Hello World!");
    }
}
```

:::tip

生成文档

```bash
javadoc -encoding UTF-8 -author -version -d comment Comments.java
```

:::
