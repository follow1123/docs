## JDK和JRE

- `JDK` (Java Development Kit) - 是 Java 程序开发工具包，包含 JRE 和开发人员使用工具
- `JRE` (Java Runtime Environment) - 是 Java 程序的运行时环境，包含 JVM 和运行时所下需要的核心库
- Java 长期支持版本 8、11、17、21
- 使用包管理器或压缩包安装
