> [官方文档](https://docs.spring.io/spring-framework/reference/core/aop/introduction-defn.html)

- `Aspect` - 切面
- `Join point` - 连接点
- `Advice` - 建议
- `Pointcut` - 切入点
- `Introduction` - 引入
- `Target object` - 目标对象
- `AOP proxy` - AOP代理
