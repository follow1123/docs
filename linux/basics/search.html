搜索

  • find 查找
    • find -name 使用文件名查找
    • find -user 使用用户名查找
    • find -size 使用文件大小查找
    • find -size +10m 表示要查找的文件大小大于10兆使用 -表示要查找的文件大小必须小于10m
  • locate 定位
  • 使用前必须使用updatedb对当前文件系统创建一个文件索引
  • 查找速度快
  • which 指令 查找那个指令在那个目录
  • grep 过滤和 |（管道命令）结合使用，参数可以使用正则表达式
    • grep -n 过滤的结果显示行号
    • grep -i 过滤的关键词忽略大小写
